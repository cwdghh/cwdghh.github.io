小白科学上网指引

发表于 2019-12-07 更新于 2020-01-03 分类于 杂事

手把手教你科学上网（利用SSR）

笔者在这里必须声明请各位科学上网时查资料就好了，不要干一些不好的事情，这样子才能长久（不然，肯定是要亏得血本无归的）。

最初笔者产生自己搭梯子的想法是因为在学业上需要看Stanford的CS231N课程，但是该课程在Coursera上并没有，因此笔者想到YouTube上看。然后，因为一时冲动，笔者在搬瓦工上买了一年49.9美刀的。。。但是很不幸的是，完全没有了解的笔者在买完了之后才发现要科学上网还需要自己搭建酸酸乳（你懂的SSR）。于是，接下来只能硬着头皮操作，希望能够别让钱打水漂。

在经历千辛万苦之后，笔者终于成功了，但是笔者发现网上所有号称零基础的面向小白的教程实际上都没有那么完整，因此笔者在这里希望能够以一个小白的身份写一篇对小白更加友好的教程，这里面可能要综合网络上的许多教程，这里就不一一列举了。

零、简单的介绍

首先，笔者希望通过简单地阐述一下（自己理解的）用SSR科学上网的原理，让小白能够大概知道自己在干什么（最好能够不要像笔者一开始一样惊慌失措）。

简单来讲，SSR能发挥作用的原因是在国外的FWQ端（这个可以在搬瓦工上购买）安装了SSR服务端，又在本地电脑或手机上安装了SSR的客户端。这样，我们的数据就可以经由国外FWQ，曲线访问国外的网站。

还有，笔者懒得放上图片了，所以建议阅读本教程是在不能理解时，问度娘吧！

一、准备国外的FWQ

这里我们仅以搬瓦工为例（因为笔者只在搬瓦工上操作过）。

如下是购买的流程：

  1. 首先我们选择需要的购买的FWQ，add to cart，再点击checkout。
  2. 因为一般是没有账户的，因此填写就是直接注册了。
  3. 付款完成之后点击右上角进入Client Area，再选择Services中的My Services按钮，进入搬瓦工的服务管理，之后选择KiwiVM Control Panel进入面板管理界面。
  4. 在这里，我们可以知道你的服务器的IP地址、端口号、选择开机或关机状态等。这里我们需要注意选择Mount ISO中挂载一下。此外，可以把你的IP地址、端口号复制粘贴到一个记事本中，在之后的步骤中会用到。
  5. 在左侧的Root password modification，然后重置一下root，注意这里需要复制粘贴记录下新的root密码（因为root密码只会出现一次），保存在一个记事本中，之后的步骤中会用到。

二、用xshell操作国外的FWQ

接下来，我们就要开始搭建国外FWQ的服务端啦！

直接上操作流程：

  1. 在xshell的官网上下载xshell，这里注意安装的时候类型选择“免费为家庭/学校”(笔者在下载时，使用edge浏览器下载失败了，用火狐浏览器才下载成功，这里的原理不明)。
  2. 打开xshell，点击“新建会话”，填入你的IP地址，端口号。
  3. 点击“用户身份验证”，然后用户名为root、密码为之前保存在记事本中的root密码。点击确定即可成功连接。

三、在本地电脑上安装客户端

这个部分其实没啥好讲的，为了防止小白们真的什么都不懂，请各位百度搜索“SSR下载”之类的，自行下载所需的SSR客户端，然后点击服务器管理，按照之前保存在记事本中的各项依次输入，点击连接，即可开始体验自由。
