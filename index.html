C语言实现归并排序与快速排序

要考试了，这里实现一下比较有意思的两种排序方法：归并排序与快速排序；

归并排序

首先我们应该知道归并排序有两种实现方式，即自下而上与自上而下。这里完成分别的实现。

归并排序的基本原理是将所要排序的数组进行不断地二分，直到形成每两个一组的情况；然后，依次完成排序。这样子排序的时间复杂度为 O(N*log N) ，当N非常大时，归并排序比选择排序和冒泡排序都要快很多。

快速排序

快速排序算法的基本想法类似于归并排序：

算法要求每一次将数列根据一个pivot划分为两个小数列，满足pivot左边的数都小于pivot，同时pivot右边的数都大于pivot。如此反复操作，直到小数列只有一个数时停止。因此，在快速排序的程序里肯定会有递归想法的运用。

容易知道，快速排序在最佳情况下的速度和归并排序一样。

C语言学习笔记

C语言学习笔记

笔者今年学习C语言，其实还是很兴奋的。作为一个几乎零基础的学习者，笔者发现自己的C语言基础简直薄弱到“令人发指”的地步，因此打算写这篇文章来记录一下自己经常忘记的知识点。

笔记

  1. 在PTA上打程序填空题一定不要忘了加上 “;” ！不要忘了，不要忘了，不要忘了！重要的事情说三遍！

  2. C是大小写敏感的，亦即APH和aph是两个不同的变量。

  3. C中常量和变量都有数据类型。
  4. C的标识符要求由数字、字母、下划线组成，并且数字不可以作为第一个(理论上是这样的，但是实际上 $ 也可以)。标识符分为关键字、预定义标识符、用户标识符。关键字不可以作为用户标识符，main、printf、scanf、define、If（首字母大写了）均可以用作用户标识符。预定义标识符可以作为用户标识符。
  5. C编写的程序称为源程序，又称编译单位。读程序都要从main{}入口， 有且仅有一个main函数作为程序的起点。
  6. 编译预处理不是C的一部分，不占运行时间，不要加分号。

  7. define PI 3.1415926，一定不可以加上分号（预定义的宏，你硬要加上分号也不是不可以，就是会把分号也替换进去，然后就会出错）。

  8. C在函数中不可以定义函数，但是可以声明或嵌套调用函数（但是在函数内部声明的函数的作用域只在被声明的函数内）。
  9. 出现if的地方不一定需要出现else。
  10. 区分C语言中不同语句的值与副作用，比如赋值语句 k = 0，如果理解了这种区别就能搞清楚为什么 while ( k = 0 ) { ; } 的循环语句一次也不会执行。
  11. 检查C语言中的 if 语句是否正确时我们要确保覆盖到了每一条分支，然后边界条件最好也要能单独分开来检测。
  12. 使用嵌套 if 语句时，C语言规定 else 总是和之前与其最近且不带 else 的 if 配对。
  13. 定义时，数组的大小不能是变量。
  14. 循环体如包括有一个以上的语句，则必须用一对大括号{}括起来，组成复合语句，复合语句在语法上被认为是一条语句。
  15. 在嵌套循环（多层循环）中, 每一层循环中都不应该改变其他层使用的循环变量的值，以免互相干扰。
  16. break 语句只能用在循环体内和 switch 语句体内。
  17. C语言中，通过函数的调用可以得到零个或者一个返回值。
  18. 不同于Pascal，在C语言中的 sizeof() 并不是函数而是一个运算符！
  19. 在C语言中，不论将几维数组传入函数，实际上都是传入数组的头指针。因此，虽然二维数组在函数定义时要指出第二维的大小，在调用时只需要输入数组名（因为此时的数组名即为数组的头指针）。
  20. 取整运算符 % 只能用在整数算子之间。
  21. 数组的下标不可以是浮点数类型。
  22. 如果初始化时指定的的元素个数比数组大小少，剩下的元素都会被初始化为 0。
  23. 全局变量与局部变量可以同名，范围小的会在 {} 所示范围内覆盖范围大的。
  24. 在所有的关系运算符（>=、>、==、!=、<=、<）中，优先级最低的运算符是 “==、!=” 。
  25. 关于C语言指针的运算：指针只有加减操作，没有乘除操作。指针可以加常数、减常数；相同类型的指针可以相减，但是不能相加。
  26. int array[10]; 则该语句定义了一个数组array，其中array是数组名，并不是整型指针。此外，对数组名是不可以用 array++，array— 之类的运算的。
  27. 在字符串数组定义时，是可以使用字面量给它初始化的，如 char str[10] = “string”; 但是数组定义完成之后就不能那字面量直接赋值了。此外，还应该注意的是 char str[10] = ‘string’ 形式的初始化是不正确的，即在C语言中要区分 “” 和 ‘’ 的区别。当然，如果是指针的话，则定义时与定义后均可以使用字面量给其赋值，当然遗憾是不可以通过下标访问对指针相应位置上的值进行修改。
  28. 区分 int (*p)[4] 和 int *p[4] ，前者指的是定义了一个指针p，这个指针指向一个有4个元素的数组，后者等价于 int *(p[4]) ，指的是定义了一个指针数组p，其中包含了4个指针作为元素。
  29. 关于C语言中的格式输入输出：

printf:

格式字符 意义
d 以十进制形式输出带符号的整数（正数不输出符号）
0 以八进制形式输出无符号的整数（不输出前缀0）
x, X 以十六进制形式输出无符号整数（不输出前缀0）
u 以十进制形式输出无符号整数
f 以小数形式输出单、双精度实数
e, E 以指数形式输出单、双精度实数
g, G 以 %f 或 %e 中较短的输出宽度输出单、双精度实数，注意这个时候输出的浮点数后置零不输出
c 输出单个字符
s 输出字符串
标志字符 意义
- 结果左对齐，右边用空格补齐，不加默认右对齐
+ 输出符号（正号或者符号）
空格 输出值为正时，前置空格；输出值为负时，前置负号
# 对于八进制数，输出时前缀0；对于十六进制数，x类前缀0x ，X类前缀0X

此外，输出的最小宽度用形如 %2d 表示输出占两个格子；形如 %02d 表示输出占两个空格，不够的位数用0补齐；形如 %.2f 表示输出小数点后两位的浮点数；形如 %.2s 表示输出两个字符，截去超过的部分。

最后，还可以在整数类型的格式字符前缀 h 或 l 分别表示输出短整型和长整型。

scanf:

格式字符 意义
d 输入十进制整数
o 输入八进制整数
x 输入十六进制整数
u 输入无符号的十进制整数
f, e 输入实型数（用小数形式或指数形式）
c 输入单个字符
s 输入字符串（遇到空格、换行、制表符等即停止读入）

此外，* 符号表示该输入项读入后不赋予相应的变量，即跳过该输入值，例如：scanf(“%d %*d %d”, &a, &b); 表示忽略输入的第二个数，把第一个与第三个分别赋给a, b；形如 %5d 表示只读入整数的前五位，例如：scanf(“%4d%4d”, &a, &b) 表示将输入的数前四位赋给a，之后四位赋给b。
